✏️ CRUD Operations — Create, Read, Update, Delete

Odoo ki jaan — inhe samjho toh ORM pe command ho jaaye

📖 What is CRUD?

CRUD = Create, Read, Update, Delete — yeh chaar operations hain jo kisi bhi database app ki backbone hote hain. Odoo mein har model ke liye yeh methods built-in hain.

💡 Note: Yeh methods ORM page mein detail mein hain. Yahan sirf quick reference aur advanced usage diya gaya hai.

CRUD Methods

Create, Write, Read, Delete

✨
.create(vals)
Naya record banata hai
WRITE
▼
🗣️ Hinglish: Naya record banana ho toh create(). Ek dict de, Odoo record bana ke return karega.
record = self.env['model'].create({'field': value})
Python
1234567
partner = self.env['res.partner'].create({
    'name': 'Rahul Sharma',
    'email': 'rahul@example.com',
})
print(partner.id)  # → 42

# ── Multiple records ──
prods = self.env['product.product'].create([
    {'name': 'Laptop', 'list_price': 50000},
    {'name': 'Mouse',  'list_price': 500},
])
📝
.write(vals)
Record update karo
WRITE
▼
🗣️ Hinglish: Pehle se existing record mein change karna ho — write(). Ek dictionary do.
recordset.write({'field': new_value})
Python
12345
partner = self.env['res.partner'].browse(42)
partner.write({'name': 'Rahul Kumar', 'phone': '99999'})

# ── Shortcut (internally write hota hai) ──
partner.name = 'New Name'
📖
.read(fields)
Data dictionary mein le aao
READ
▼
🗣️ Hinglish: Record ko dict mein chahiye (API response ya debug) toh read(). List of dicts return karta hai.
Python
123
data = partner.read(['name', 'email'])
# → [{'id': 42, 'name': 'Rahul', 'email': '...'}]
🗑️
.unlink()
Record delete karo
DELETE
▼
🗣️ Hinglish: Permanently delete. Ek baar gaya toh wapas nahi.
Python
12
partner.unlink()
# Multiple bhi: partners.unlink()
📋
.copy(default)
Duplicate banao
WRITE
▼
Python
1234
new_order = order.copy()  # Simple duplicate
new_order = order.copy(default={
    'client_order_ref': 'COPY-001'
})
📦
Multi-Record Operations
Ek saath multiple records pe kaam karo
TRICK
▼
🗣️ Hinglish: write() aur unlink() directly multi-record pe kaam karte hain. create() list of dicts se multiple bana sakta hai.
Python
12345
# ── Update all draft orders ──
drafts = self.env['sale.order'].search([('state',=,'draft')])
drafts.write({'note': 'Auto-reminder sent'})

# ── Delete all inactive partners ──
inactive = self.env['res.partner'].search([('active',=,False)])
inactive.unlink()
